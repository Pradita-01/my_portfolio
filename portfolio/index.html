Projects

Aadhaar Fake Detection using CLIP & PyTorch

Situation: With increasing incidents of forged Aadhaar cards being used for fraudulent activities in India, there was a need for an automated system to validate Aadhaar authenticity.
Task: Design and implement a robust machine learning system capable of detecting fake Aadhaar cards using multimodal inputs (text + image).
Action: Utilized OpenAI’s CLIP model to extract joint text-image embeddings, trained a classifier on synthetically generated real and fake Aadhaar card samples using PyTorch, and fine-tuned the model using pretrained weights for better convergence.
Result: Achieved 80% validation accuracy on the test set, showcasing the system's potential for real-world deployment in digital KYC pipelines.

Real-Time Cryptocurrency Tracker

Situation: Cryptocurrency markets are highly volatile, and users need real-time tools to track live prices, trends, and volumes across exchanges.
Task: Develop an interactive and responsive dashboard that streams live crypto price updates with visual insights.
Action: Built a full-stack web application using React for the frontend, Redux Toolkit for state management, and integrated real-time WebSocket APIs to fetch and update cryptocurrency data dynamically.
Result: Delivered a seamless dashboard with auto-refreshing price charts, high responsiveness, and accurate real-time data — improving user experience for crypto traders.

GLOF Prediction & Early Warning System

Situation: Glacial Lake Outburst Floods (GLOFs) pose severe risks to Himalayan regions due to sudden lake breaches triggered by climate change and glacial melt.
Task: Build a scalable early warning system that predicts potential GLOFs and provides timely alerts to local authorities.
Action: Developed a hybrid solution integrating real-time IoT sensor data (temperature, water level), LSTM-based time-series prediction models, and a Django-based web dashboard to issue warnings and visualize risk levels.
Result: Successfully simulated early warning capabilities with predictive insights, contributing to disaster mitigation and preparedness strategies.

Secure File Encryption System

Situation: Handling large sensitive files securely is a critical requirement in operating systems, especially in file transfers and data storage.
Task: Create an efficient command-line tool that encrypts and decrypts files using strong encryption standards.
Action: Developed a C-based CLI tool using OpenSSL libraries that implemented AES-256-CBC encryption with secure initialization vector (IV) randomization and file chunking to handle large files without memory overflow.
Result: Delivered a secure, efficient, and lightweight file encryption system ensuring confidentiality, usability, and robust protection against brute-force attacks.
